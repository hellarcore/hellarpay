  Wie man hellars abbaut?
Bergbau Hellar ist ein anspruchsvolles Geschäft, das technische Fähigkeiten und Kenntnisse erfordert. Wenn Sie sich entschließen, Hellar abzubauen, können Sie die Details lesen und Fragen stellen unter Fragen stellen auf Community-Forum.

In Ihrer Geldbörse ist eine Funktion zum Schürfen von Münzen integriert. Hier sind einfache Techniken für Anfänger, wie man sie benutzt: 

1. Windows
Erstellen Sie nach der Installation der Brieftasche eine Verknüpfung mit der Exe-Datei программы (hellar-qt.exe). Klicken Sie mit der rechten Maustaste auf die Verknüpfung und öffnen Sie "Eigenschaften". Im Feld "Objekt". geben Sie ein Leerzeichen ein -gen. Speichern Sie die Änderungen und starten Sie die Verknüpfung. Wenn die Brieftasche bereits gestartet wurde, schließen Sie sie, bevor Sie die Verknüpfung starten. Starten Sie die Verknüpfung, schließen Sie sie.

Es werden keine Indikatoren folgen. Der Beweis dass der Algorithmus läuft, ist eine 100%ige CPU-Auslastung. Dann müssen Sie nur noch warten, bis neue hellars eintreffen auf eine aktive Internetverbindung.

2. MAC OS
Installieren Sie die Brieftasche und schließen Sie sie, wenn sie geöffnet wurde geöffnet wurde.
Um mit dem Mining zu beginnen, müssen Sie als nächstes "Terminal". Starten Sie das Terminal Programme > Dienstprogramme > Terminal.app
 
Rufen Sie den Ordner wallet mit dem Befehl: 
cd hellar

Start daemon: 
./hellard -daemon

Start des Bergbaus: 
./hellar-cli set generate true

Beim ersten Start werden Sie außerdem aufgefordert, Zugriffsparameter für rpc-Anfragen zu ~/.hellar/hellar.conf hinzuzufügen; ein Beispiel für diese Parameter wird dort gezeigt. Sie können auch daemon=1 in der Konfiguration angeben, so dass Sie diesen Parameter nicht jedes Mal angeben müssen, wenn Sie hellard starten.

Zum anhalten: 
./hellard stop

Sie können die Bergbauinformationen mit dem Befehl: 
./hellar-cli getmininginfo

Nach dem Start sie müssen nur darauf warten, dass neue Gsbs mit einer aktiven internetverbindung eintreffen..
