Network «Hellar» unicentred  calculations. Information about transactions is fully distributed among users and is stored and transfers are made directly from user to user.» (P2P). here are some peculiarities associated with this that need to be known and correctly applied: 

  • P2P Transaction Hellar transfer is created by the sender, it contains the unique address of the recipient and the amount. of the recipient and the amount. Then data is encrypted, shall be signed with the secret key held by the Sender, and are being sent to the network. The recipient of the transfer decrypts the data using its secret key and credits the amount to self balance after several confirmations from other members of the network, indicating the authenticity of the transaction. Attention!!! All transactions are irrevocable, i.e. there is no possibility of cancellation!
  • Addresses – these are unique accounts on which Hellar is accounted for (have approximately the following form: HhaSqxvF69ph85aDUqph1pBViW7zBYnKRz). Anyone can create an unlimited number of addresses and Use them for one-time operations or for long-term storage. Addresses are also used as as an identifier when sending and receiving Hellar. That is the transfer is made from Address to Address.

  How to mine hellars?
Mining Hellar – is a demanding endeavour that requires technical skills and knowledge. If you have decided to mine Hellar, you can read the details and ask questions on community forum.

There is a coin mining feature built into your wallet. Here are simple tricks for beginners on how to use it: 

1. Windows
After installing the wallet, create a shortcut to the exe file programme (hellar-qt.exe). Right-click on the shortcut and open "Properties". In the "Object" field. enter a space -gen. Save the changes and run the shortcut. If the wallet has been launched, close it before launching the shortcut. close it before launching the shortcut.

No indicators will follow. The evidence that the algorithm is running will be 100% CPU utilisation. Next, you just have to wait for new Hellar to arrive with an active Internet connection.

2. MAC OS
Put the wallet back and close it if it has been open.
Next, to start mining you need to use "Terminal." Start the terminal Programmes > Utilities > Terminal.app
 
Access the wallet folder with the command: 
cd hellar

Launch daemon: 
./hellard -daemon

Run mining: 
./hellar-cli set generate true

On first run it will also ask you to add access parameters for rpc requests to ~/.hellar/hellar.conf, an example of these parameters will be shown there. You can also specify daemon=1 in the config, so that you don't have to specify this parameter every time you start hellard.

To stop: 
./hellard stop

You can view mining information with the command: 
./hellar-cli getmininginfo

Once launched you just have to wait for new hellars to arrive with an active internet connection.
